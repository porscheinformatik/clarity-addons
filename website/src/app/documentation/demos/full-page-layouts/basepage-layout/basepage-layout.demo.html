Template

  • Select the pagelayout and detach it from the main-component to edit the content layer.
  • Add ui elements into the content-area.
  • Using the Autolayout functionality is recommended.
  • See properties of page-title component, toggle the history, command-bar, and action-panel button as required.

Content Panel

  • Use the content panel to provide supplementary content without interrupting the main content
  • If just actions are required use the action-panel instead
  • Multiple content panels might be called from within a page (eg. offers, financing, ... referring to a vehicle in the main content)

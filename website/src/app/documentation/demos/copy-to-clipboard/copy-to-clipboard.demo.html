A generic copy-to-clipboard button that can be placed inside any container (datagrid cells, treetable cells, table cells, or summary items) to let users copy a value with a single click.

The component uses the Angular CDK clipboard module to perform the copy operation. It displays a copy-to-clipboard icon that changes to a success icon for 1 second after a successful copy.

When used inside data cells, set [hiddenUntilHovered]="true" to keep the icon hidden until the user hovers over the parent element. This avoids visual clutter in dense grids while still providing easy access to the copy action.

Note: The summary area's [valueCopyable] input on clr-summary-item uses this component internally.

Summary of Options

Input Type Default Description
[value]
Type: string (required)
 string required The text value to copy to the clipboard when the icon is clicked.
[tooltipText]
Type: string
Default: 'Copy to clipboard'
 string 'Copy to clipboard' The text displayed in the tooltip when hovering over the copy icon.
[hiddenUntilHovered]
Type: boolean
Default: false
 boolean false When true, the copy icon is hidden until the user hovers over the parent element (e.g. a table cell). Ideal for use in datagrids, treetables, and tables to reduce visual noise.

Code & Examples

For inline alignment of the value and the copy icon, add the clr-copy-to-clipboard-container class to the direct parent element of <clr-copy-to-clipboard>. This applies display: flex !important, flex-direction: row and align-items: center to keep the text and icon on the same line.

Datagrid

Place <clr-copy-to-clipboard> inside a <clr-dg-cell>. The icon appears when hovering over the cell.

ID Name Email @for (user of users; track user.id) { {{ user.id }} {{ user.name }} {{ user.email }} } {{ users.length }} users

Treetable

Place <clr-copy-to-clipboard> inside a <clr-tt-cell>.

Department Code @for (dept of treeData; track dept.code) { {{ dept.name }} {{ dept.code }} @for (child of dept.children; track child.code) { {{ child.name }} {{ child.code }} } } No departments found

Plain HTML Table

Works in standard <table> elements as well. The parent hover detection targets the immediate parent element (the <td>).

In plain HTML tables, an additional justify-content: center style is needed on the cell to match the default table cell alignment. Add the utility class clr-justify-content-center alongside clr-copy-to-clipboard-container on the <td>.

The Image Gallery displays a set of images in a compact static view and allows the user to browse them in a full carousel modal.

Summary of Options

Input Type Default Effect
images
Type: ClrImageGalleryImage[]
Default: []
 ClrImageGalleryImage[] [] Array of image objects with src and optional alt fields.
productName
Type: string
Default: ''
 string '' Title shown in the carousel modal header.
spotlightSize
Type: number
Default: 400
 number 400 Pixel size of the spotlight square in the static view. Thumbnails scale automatically to spotlightSize / 4.

Code & Examples

Default (7 images, spotlight 400 px)

Click the gallery to open the carousel modal. Use the arrow buttons or the thumbnail strip to navigate. Close via the ✕ button or by clicking outside the modal.

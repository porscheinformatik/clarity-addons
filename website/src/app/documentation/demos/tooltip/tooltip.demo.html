Clarity Addons extensions for the native Clarity clr-tooltip component.

As the native Clarity tooltip component only supports fixed widths, the clr-tooltip-fit-content css class can be used to set the tooltip bubble's width to fit its content instead. It simply applies width: fit-content !important; to the element, making the bubble's width adjust to the text width. Note that this class is restricted to the clr-tooltip-content element exclusively.

Real-world usage in clr-copy-to-clipboard

The clr-copy-to-clipboard component applies clr-tooltip-fit-content class internally on its own clr-tooltip-content, so the "Copy to clipboard" bubble is always sized to its text rather than stretching to a fixed Clarity size. No extra input is needed on the component itself.

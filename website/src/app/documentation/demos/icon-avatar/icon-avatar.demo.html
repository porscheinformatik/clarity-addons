The icon avatar is used to display an avatar containing a Clarity icon.

Summary of Options

Input Values Default Effect
clrShape
Type: string
Default: 'user'
 String undefined The shape of the Clarity icon that gets displayed
clrSize
Type: Number
Default: 24
 Number 24 Defines the size of the icon avatar in px

Code & Examples

Standard behaviour

The icon avatar gets a hover effect when hovering over anything inside his direct parent. See it in action by hovering over the name. Default icon shape is "user".

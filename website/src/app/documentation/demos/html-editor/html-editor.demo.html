Html-Editor is a tool for creating and editing HTML/Rich Text.

Usage

Use the html-editor whenever you want to let the user compose rich formatted text in HTML form.

Screenshot of the html-editor with some text.

Summary of Options

clr-html-editor component

Input Values Default Effect
[clrHtmlContent] String '' Provide the HTML Editor with an initial HTML template.
[clrConfig] AngularEditorConfig Default Config Config for the html editor
(clrHtmlChanged) String Emits when the value changes.

Code & Examples

clr-html-editor component

To use the HTML Editor, use the clr-html-editor component and provide it with a default template. Then, listen to it's (clrHtmlChanged) event.

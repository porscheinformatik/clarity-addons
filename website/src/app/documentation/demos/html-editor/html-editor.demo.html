HTML Editor is a tool for creating and editing HTML/Rich Text.

Installation

Clarity Addons provides clarity specific custom styles for the HTML editor component. The component itself is not included itself in the clarity addons.


  1. First install through npm:
  2. Next, verify that the 'clarity-addons' style is included in the global styles of your app or in the angular.json.
  3. Last, add a CSS class "clr-html-editor" on the Angular editor component, and you're good to go!

Usage

Use the HTML editor whenever you want to let the user compose rich formatted text in HTML form.

The brand avatar is used to display logo of a brand.

Summary of Options

Input Values Default Effect
clrBrand
Type: string
Default: undefined
 String undefined The brand of the logo that should be displayed. (Case insensitive, spaces, dashes & underscores will be ignored)
clrSize
Type: Number
Default: 24
 Number 24 Defines the size of the brand avatar in px

The letter avatar is used to display initials of a name.

Summary of Options

Input Values Default Effect
clrName
Type: string
Default: undefined
 String undefined The full name from which the initials are generated
clrSize
Type: Number
Default: 24
 Number 24 Defines the size of the letter avatar in px

Code & Examples

Standard behaviour

The letter avatar gets a hover effect when hovering over anything inside his direct parent. See it in action by hovering over the name.

Larger size

For a larger size of the avatar set the clrSize attribute and adapt the font-size of the avatar accordingly.

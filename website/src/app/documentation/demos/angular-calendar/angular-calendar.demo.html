A calendar component that can display events on a month, week or day view.

Installation

Clarity Addons provides clarity specific custom styles for the angular calendar component. The component itself is not included itself in the clarity addons.


  1. First install through npm: or
  2. Next include the CSS file in the global styles of your app or in the angular.json.
  3. Verify that the 'clarity-addons' style imports come after the 'angular-calendar' style imports in your project. This sequence is crucial as it allows 'clarity-addons' styles to effectively override the 'angular-calendar' styles.

Code & Examples

mwl-calendar-week-view component

To use the weekly view of the calendar, use the mwl-calendar-week-view component. Provide a [viewDate] and [events] data to get started.

To customize for example the style and the format of day and time, utilize custom templates. For instance, if you want to display the day and date in a unique format in the header:

{{ day.date | date : "EEE" }}
{{ day.date | date : "dd" }}

mwl-calendar-month-view component

To use the monthly view of the calendar, use the mwl-calendar-month-view component.


mwl-calendar-day-view component

To use the daily view of the calendar, use the mwl-calendar-day-view component.

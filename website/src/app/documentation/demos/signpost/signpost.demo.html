A signpost component designed for use in datagrids or treetables to prevent overlapping elements.

Summary of Options

Input/Output Values Default Effect
[targetAnchor]
Type: String
Default: undefined
 String undefined Determines the location where the signpost content is rendered. For datagrids or treetables, attach it to the end of the content container to ensure proper layout and avoid overlapping elements.
[position]
Type: String
Default: right-bottom
 String right-bottom Defines the direction in which the signpost content is displayed, following the standard Clarity Design positioning system.

A signpost component designed for use in datagrids or treetables to prevent overlapping elements.

Parameter

  • targetAnchor: Specifies where the signpost content container should be attached. For datagrids or treetables, this should be placed at the end of the content div container to prevent layout issues.
  • position: Defines the direction in which the signpost content is displayed, following the standard Clarity Design positioning system.

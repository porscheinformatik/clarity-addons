The *clrIfWarning directive can be used with a clr-control-helper container to include a warning icon next to the form control and will set the color of the warning text accordingly. It can be used in combination with all control types where the clr-control-error can be used.

Usage

  • Add the *clrIfWarning to a clr-control-helper with a boolean variable to display the warning text and icon when required.
  • Currently, the Clarity Addons clr-date-time-container is not supported.
  • When using in combination with clr-combobox-container , the order of the warning and error icons changes depending on which is displayed first.

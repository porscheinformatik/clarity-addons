Internationalization

Internationalization issues should be solved consistently throughout all applications - there is no necessity to start creative achievements in this area.

Basically:

  • Date & Time,
  • Number, and
  • Currency

are affected by language and locale settings.

Please find in the following the definitions and some samples:

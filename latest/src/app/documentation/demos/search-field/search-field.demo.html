The Seach Field is used to input a value to search for it.
Hitting the delete icon will remove the current value from the input field

Best Practices

  • Placeholder: Use a Placeholder displaying common search terms.
    Limit your hint to just a few words, otherwise you actually increase the cognitive load.
  • Results: Display a message containing the searchterm, when no search results have been found.

Code & Examples

Because the search field is a directive, it can be applied on any input element. The name of the directive is clrSearch. This makes the search field compatible with the existing ClrInputContainer and forms.

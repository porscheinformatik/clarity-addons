The progress spinner of Clarity Addons extends the default progress spinner by an Angular Component which allows the developer to toggle the loading state depending on a boolean input.

Summary of Options

Input Values Default Effect
clrShowSpinner
Type: boolean
Default: false
 true, false false Controls if the spinner is shown
clrSize
Type: String
Default: "sm"
 sm, md, lg "sm" Defines the size of progress spinner

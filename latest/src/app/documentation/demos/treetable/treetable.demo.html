The Treetable component is a wrapper around a datagrid which allows for nestable rows (styling is done for up to 6 levels of nesting). It is used to represent a graph structure of data in a table fashion.

Functionality

The Treetable retains similar functionality to the datagrid style-wise, but omits features like lazy-loading, sorting, filtering, and so on.

Momentarily, the limit for nesting is 6 due to styling constraints.

No horizontal scrolling

For examples with a small number of columns but with a lot of text in each column, you might want to enable collapsing the content of the columns even further and prevent the horizontal scrollbar from appearing. Beware that enabling this option might lead to styling issues on tables with a large number of columns and/or deeper nesting.

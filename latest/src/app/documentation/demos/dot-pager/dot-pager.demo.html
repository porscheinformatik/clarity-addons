The Dot-Pager is a basic component that can be used to show a simple pagination bar.

Usage

  • Use the Dot-Pager component whenever you have to show data over multiple pages, e.g. a list of Cards.
  • Do not use the Dot-Pager component when you are showing your data in a clr-datagrid, instead use the existing clr-dg-pagination component.

Code & Examples

Following interactions are available:

  • To define the pages add a [clrPages] input to the component.
  • To define the current page add a [clrCurrentPage] input to the component.
  • To react to page changes, bind to the event (clrCurrentPageChange).

Max Width

The css class clr-max-width can be used to set the max-width. It is set 65 rem and used for View/Edit sections as a default.

Use when

Your page content never needs more width and starts looking awkward in case it's getting more width.

Don't use when

Your datagrid has various widths because of the column show/hide feature and when all colums are enabled,
the additional width would be beneficial for the user.

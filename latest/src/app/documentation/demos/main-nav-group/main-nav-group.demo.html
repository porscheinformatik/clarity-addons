The main navigation group is used to define a collapsible menu in the main menu bar. It also fits into the responsive behaviour of the standard main menu, which will move to the hamburger menu on small screens.

Code & Examples

Following interactions are available:

  • To define the title of the group add a [clrTitle] input to the component.

Its crucial to set routerLinkActive="active" on the component itself, to highlight the title as soon as a sublink is active.

On the links itself the class nav-link has to be set for correct styling. Furthermore routerLinkActive="active" should be set to highlight the links in the menu when they are active.

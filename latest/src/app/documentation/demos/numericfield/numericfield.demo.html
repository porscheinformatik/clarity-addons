The Numeric Field is used to enter numeric values of a certain type.

Usage

  • Invalid characters are ignored during input.
  • Amount of decimal places should be defined according to the input type and usecase.
  • Decimal separator and grouping separator should be set according to locale.

Use when

  • One of the following inputs is required from the user:

Don't

Code & Examples

Because the numeric field is a directive, it can be applied on any input element. This makes the numeric field compatible with the existing ClrInputContainer and forms. See code examples below which include clr-control-error and clr-control-helper elements.

The numeric field requires you to use custom validators, namely ClrNumericFieldValidators to validate min and max values.

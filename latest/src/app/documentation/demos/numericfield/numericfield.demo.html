The NumericField component is used to force the user to enter a numeric value. Invalid characters are ignored during input.

Angular Component

Summary of Options

Input Values Default Effect
clrLabel
Type: String
Default: ""
 Javascript String "" Label to be shown for input field.
clrDecimalSep
Type: String
Default: ","
 Javascript String "," Defines the decimal separator.
clrGroupingSep
Type: String
Default: "."
 Javascript String "." Defines the grouping separator.
clrDecimalPlaces
Type: Number
Default: 2
 Javascript Number 2 Defines the number of allowed decimal places.
clrNumericValue
Type: Number
Default: ""
 Javascript Number "" Two-way bound parameter for the value of the input component.
clrNumericValue: {{amount}}

The NumericField directive is used to aid the user when entering a numeric value. Invalid characters are ignored during input. Optionally the numeric field can also display a unit.

Because the numeric field is a directive, it can be applied on any input element. This makes the numeric field compatible with the existing ClrInputContainer and forms including validation in general. See code examples below which include clr-control-error and clr-control-helper elements.

Type: string
Default: "right"

Breadcrumbs are used to show the hierarchical (not historical) path to the actually viewed page. Don't use it as a browsing history.
  1. Page before last
  2. Last page
  3. Actual Page
Don't

Don't use it as browsing history.

Do

Use it as hierarchical path to the actually viewed page.

Code & Examples

To set the breadcrumbs, the service ClrBreadcrumbService in conjunction with ClrBreadcrumbModel has to be used.

To display the breadcrumbs simply place the angular component anywhere needed.

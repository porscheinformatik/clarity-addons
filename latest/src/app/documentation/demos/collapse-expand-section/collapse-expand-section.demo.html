The Collapse/Expand Section is a basic component that can be used to switch between hiding & showing content anywhere on your page.

Usage

Use the Collapse/Expand Section whenever you want to show specific data only when a certain condition is met, e.g. the user deciding to open the section. Useful for hiding detail information.

Code & Examples

Following interactions are available:

  • By setting the [clrIsCollapsed] input of the component, you can define if your section is collapsed or expanded when first opening the page. Default is true.
  • (clrCollapsed) is fired every time the component collapses.
  • (clrExpanded) is fired every time the component expands.
  • You can optionally add a subtitle by adding any HTML with the attribute clr-ces-subtitle, similar to the title.

The Base Pagelayout provides the most fundamental navigation and content elements.

Design Guidelines

All pages in an application should follow a common layout structure to ensure a consistent user experience across applications.

The Base Pagelayout consists of:

  • Main Navigation
  • Pagetitle
  • Content Area

Futher components may complete the pagelayout as needed:

  • Command bar
  • Sidebar
  • Content-Panel

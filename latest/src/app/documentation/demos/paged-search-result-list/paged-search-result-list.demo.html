The Paged Search Result List component is a basic component used to display a list of generic data including a pager above and below the list.

Style

You can style the list any way you want - this component allows you to use a custom template to show all items of your list. The custom template can range from a single line representing text to one or more completely new angular components.

Code & Examples

To set a template for each item of your list simply add a ng-template inside the clr-paged-search-result-list. By defining let-item="item" on your ng-template you can get the context of each individual item.

Following interactions are available:

  • To show items add a [clrItems] input to the component.
  • To set the current page add a [clrPage] input to the component.
  • To define the page size add a [clrPageSize] input to the component.
  • To define the number of total items add a [clrTotalItems] input to the component.
  • To set the position of the pager add a [clrPagerPosition] input to the component. Default is bottom
  • To react to page changes, bind to the event (clrPageChange).

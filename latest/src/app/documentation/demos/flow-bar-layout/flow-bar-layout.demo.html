The Flow Bar is a basic component that can be used to highlight the current progress a user is making throughout multiple steps. The Flow Bar also allows the user to navigate between steps, either by clicking on a specific step or selecting the previous/next buttons.

Usage

When implementing a flow bar on your page, you will want to extend your div with class content-header with a clr-flow-bar.

Following inputs & outputs are available:

  • [clrSteps] - an array of FlowBarStep. You can extend/implement this interface to save even more information in your steps.
  • [(clrActiveStep)] two way binding of the active step.

You can use the @ViewChild() annotation to get a reference of your ClrFlowBar, which will allow you to call its methods:

  • previous() - open the previous step.
  • next() - opens the next step.
  • isPreviousAvailable() - returns true if the previous step can be opened.
  • isNextAvailable() - returns true if the next step can be opened.
  • isLastStep() - returns true if the current step is the last.

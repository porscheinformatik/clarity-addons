The original card documentation can be found in the official Clarity Documentation. This document only extends the clarity documentation by two added class: card-active and selectable.

Usage

Use the card-active class when you have a list of cards and want to show which card is currently selected. Every selectable card should have the class selectable instead of the clarity-default clickable class.

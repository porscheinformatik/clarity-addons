Keyboard Navigation

Two standalone directives add keyboard-driven navigation to any Clarity navigation component (clr-tabs, clr-vertical-nav, or any custom host).

1. clrKeyboardNavCtrlArrow

Add clrKeyboardNavCtrlArrow to a navigation host. While focus is anywhere on the page, use Ctrl+Left / Ctrl+Right to cycle through tabs (or Ctrl+Up / Ctrl+Down for a vertical nav). Navigation wraps around at both ends.

Try it: press Ctrl+Right / Ctrl+Left to switch tabs.

2. clrKeyboardNavAltMnemonic

Add clrKeyboardNavAltMnemonic to a navigation host. Numeric mnemonics (1, 2, 3, …) are assigned to each item at page load and never change. Hold Alt to reveal the badge overlays, then type the digit(s) and release Alt to navigate.

Multi-digit test: Alt held → 1 → 0 → Alt released navigates to item 10. There are 11 tabs below so both Alt+10 and Alt+11 require two keystrokes.

Try it: hold Alt to see the badges, then press a digit (or two).

3. Both directives combined

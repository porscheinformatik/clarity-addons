Image Gallery in Modal Demo

Fixed image size ([imageSize]="400")

The gallery is embedded inside a Clarity modal with [useBuiltInModal]="false". Clicking an image replaces the modal content with the carousel instead of opening a nested modal. The carousel uses explicit pixel sizes.

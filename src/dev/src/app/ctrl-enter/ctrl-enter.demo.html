Ctrl+Enter Directive

The clrControlEnterSubmit directive enables form submission by pressing Ctrl+Enter (or Cmd+Enter on macOS) while focus is within any input field of the form.

1. With Tooltip (Pretranslated Text)

This form uses [clrControlEnterSubmit]="'Ctrl+Enter to submit'". The tooltip text is pretranslated and passed directly to the directive. Hover over the submit button to see the tooltip.

2. Without Tooltip

This form uses clrControlEnterSubmit without a value (or with an empty value). No tooltip is displayed on the submit button.
